# spif.amsterdam

Personal artist site for **Spif** — Amsterdam-based artist working at the intersection of live electronic music, ritual performance, AI-mediated installation, and tactile interactive objects.

## Works

- **Vox Futuri** *(in development, 2026)* — AI-mediated future-self oracle. Vintage cockpit TV + stripped Soviet tank radio + 2-way mirror over LCD + camera + Raspberry Pi. *Submitted for Nxt Museum's Night Academy residency.*
- **Vox Ignis / Diepgang** *(2026)* — 90-min spoken-word + live electronic ritual. Premiering at WAAN/ZIN Festival, De Lievelinge, May–June 2026.
- **Vox Arboris** *(2023, 2026)* — Live modular electronic performance + gallery installation form. A'dam Tower (Apr 2026); WIHH gallery month-long residency (Summer 2023).
- **Meditation Station** *(2025)* — Tactile meditation installation in 1950s bakelite TV. Raspberry Pi + dual rotary encoders.
- **Cubic Reset — Unscramble Your Life** *(2015)* — 3×3 metre interactive installation. Brave Festival (~5000 attendees), Tuinen van West, Amsterdam.
- **Find Your Inner Unicorn** *(2014)* — Interactive installation in a hand-painted vintage caravan. Brave Festival, Tuinen van West, Amsterdam.
- **Tardis** *(year TBC)* — 20-minute participatory psychological installation. Recreated the TARDIS as a time machine; participants stepped 5 years into the future to converse with the self that had already achieved their goal. Created and built with a 5-person team; performed with hired actors. The Next Web Conference, Amsterdam. *Conceptual prototype for Vox Futuri.*
